Spring 介绍

Spring是一个轻量级框架。不依赖于容器就可以运行。Struts，Hibernate 同属于轻量级。
Spring以IOC，AOP为主要思想

  • IOC：Inversion of Control， 控制反转。
  • AOP：Aspect Oriented Programming， 面向切面编程。通俗意思就是在执行某些代码前执行另一些代码，可以做出添加，删除某些功能。类似于Servlet中的Filter，思想相同。

Spring 属于非侵入式

  • 侵入式：通常需要通过实现特定的接口，继承特定的类才能增强功能
  • 非侵入式：对现有类的结构没有影响，就可以增强 JavaBean 的功能

Spring 三层构架

  1. 表现层(Web): Spring MVC
  2. 业务逻辑层（Service）：Spring的IOC
  3. 数据访问层（DAO）：Spring的jdbc

使用Spring重要的jar包

四个必须包

  1. spring-core：依赖注入IoC与DI的最基本实现
  2. spring-beans：Bean工厂与bean的装配
  3. spring-context：spring的context上下文即IoC容器
  4. spring-expression：spring表达式语言

aop部分包含4个模块

  1. spring-aop：面向切面编程
  2. spring-aspects：集成AspectJ
  3. spring-instrument：提供一些类级的工具支持和ClassLoader级的实现，用于服务器
  4. spring-instrument-tomcat：针对tomcat的instrument实现

test部分只有一个模块，我将spring-context-support也放在这吧

  1. spring-test：spring测试，提供junit与mock测试功能
  2. spring-context-support：spring额外支持包，比如邮件服务、视图解析等

commons-logging: 日志（必须包）

Spring Bean 三种不同配置方式

基于 XML 配置方式

导入 jar 包

  1. commons-logging
  2. spring-aop
  3. spring-beans
  4. spring-context
  5. spring-core
  6. spring-expression

  7. spring-test

    创建 Bean

    package com.carlos.spring;

    public class helloword {

    private String title;
private String name;

public helloword(String title, String name){
    this.title = title;
    this.name = name;
}

public void show(){
    System.out.print(title + ":" + name);
}

    }

创建测试类

package com.carlos.spring;

public class test {
    public static void main(String[] args) {

        ApplicationContext hw = new ClassPathXmlApplicationCont("ApplicationContext.xml");

        helloword h = (helloword) hw.getBean("hello");

        h.show();

    }
}

创建 XML 文件

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xmlns:c="http://www.springframework.org/schema/c"
xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

<bean id="hello" class="com.carlos.spring.helloword">
    <constructor-arg name="title" value="hello word"></constructor-arg>
    <constructor-arg name="name" value="carlos"></constructor-arg>
</bean>
</beans>

使用构造器注入

  • 添加xmlns:c="http://www.springframework.org/schema/c"
  • <constructor-arg name="" value=""></constructor-arg>
  • Spring c- 命名空间`<bean id=”hello” class=”com.carlos.spring.helloword”
    c:name="carlos"
c:title="hello word!">
    `
  • XML 不容许数字作为属性的第一个字符，想要使用需要加一个 _ 前缀

使用 Set 方法注入

  • 添加xmlns:p="http://www.springframework.org/schema/p"
  • <property name="" value=""></property>
  • Spring p- 命名空间`<bean id=”hello” class=”com.carlos.spring.helloword”
    p:name="carlos"
p:title="hello word!">
    `

装配集合

使用 List 元素申明一个列表

<property name="num">
    <list>
        <value>第一</value>
        <value>第二</value>
        <value>第三</value>
    </list>
    </property>
  • 与 性质相同，都可使用。不同的是，set 不可存放重复数据，存放顺序同样不可保证。推荐使用 。不论在哪种情况下俩种集合都可使用。

基于注解的配置方式

注解

  • @Component : 表明该类会作为组件类。
  • @ComponentScan : 能够在Spring中启用组件扫描。如果没有其他的配置，默认会扫描与配置类相同的包。
  • @Controller : 用于对 Controller 实现类进行标注，功能与 Component 相同。
  • @Repository : 用于对 DAO 实现类进行标注。
  • @Service : 用于对 Service 实现类进行标注，功能与 Component 相同。

创建 Bean

DAO 层

使用注解 @Repository

@Repository
public class UserDao {
    public void save(){
        System.out.println("数据库");
    }
}
  • 等效于 XML 中 <bean id="userDao" class=" ... "/>
  • ID 为 userDao 而不是 UserDao,将类名第一个字母变为小写。

Service 层

使用注解 @Service

@Service
public class UserService {
    public void save(){
        System.out.println("service");
    }
}

Servlet 层

使用注解 @Controller

@Controller
public class UserServlet {
    public void save(){
        System.out.println("servlet");
    }
}

创建 XML 启动组件扫描

<beans ...>
    <context:component-scan base-package=" 包名 "></context:component-scan>
</beans>

创建测试类

public class Test {
    public static void main(String[] args) {
        ApplicationContext act = new ClassPathXmlApplicationContext    ("ApplicationContext.xml");

        UserDao ud = (UserDao) act.getBean("userDao");
        ud.save();

        UserService user = (UserService) act.getBean("userService");
        user.save();

        UserServlet us = (UserServlet) act.getBean("userServlet");
        us.save();
    }
}

基于 Java 类的配置方式

  • Bean ： 声明 Bean

    创建配置类

@Configuration
public class User {
    @Bean
    public String show(){
        return "hello word!";
    }
}

@Configuration
public class Student {
    @Bean
    public String show1(){
        return "student";
    }
}

创建测试类

public class Test {
    public static void main(String[] args) {
        AnnotationConfigApplicationContext at = new AnnotationConfigApplicationContext(User.class);
        System.out.println(at.getBean("show"));

        AnnotationConfigApplicationContext stu = new AnnotationConfigApplicationContext(Student.class);
        System.out.println(stu.getBean("show1"));
    }
}
